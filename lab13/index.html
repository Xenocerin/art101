Lab 13 - Loops

Challenge

The challenge was to take user input in the form of factors and the word to print when that factor is present in a listed number, it then begins listing numbers up to a maximum that is also input by the user. It does this via a function that takes the user's maximum number and an array storing the user's factors. It then uses a loop which repeats starting a zero until it reaches the maximum number with a nested loop that runs through the array and contains an if statement that divides the current number and checks if it equals zero, if so it prints the associated keyword.
