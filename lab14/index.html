Lab 14 - Debugging Tools & Strategies

Challenge

The challenge was to go back and correct any bugs in previous code, in my case that'd be my lab 9 toggle class button.

Problems

The issue was that despite me beleiving I properly connected the classes I was unable to toggle no the special class and couldn't figure out why. However given the number of labs and experience gained since then I was almost immediately able to diagnoise the problem as me being an idiot, turns out that since I'd made the original div and buttons classes both so similar I'd confused the div and button classes and plugged the div class into the ".click" method meaning that the button was connected to nothing. Additionally despite making a class for my buttons I listed it as an id meaning even if I would've gottent the name correct intially it wouldn't have worked. Thankfully now having a little more experience the problem was easy to recgonize and address.

Reflection

I've since realised to pay closer attention to syntax as a slight slip up can have you being forced to peer over lines of code for one spelling mistake.
